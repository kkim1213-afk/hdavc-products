OVERVIEW

TRS 컨택 배치
TRS 플러그는 팁(Tip) · 링(Ring) · 슬리브(Sleeve) 3개 접점을 갖습니다. 스테레오 용도에서는 Tip = 좌측, Ring = 우측, Sleeve = GND가 일반적이며, 밸런스드 모노 용도에서는 Tip = HOT(+), Ring = COLD(–), Sleeve = GND(쉴드)로 배선합니다.

주요 특징
KEY FEATURES

AD109 1/4인치 TRS 스테레오 플러그의 주요 특징입니다.

🎧
스테레오 / 밸런스드 겸용
TRS 3컨택 구조로 스테레오 헤드폰 케이블, 밸런스드 모노 라인 케이블 어느 용도로도 결선 가능합니다. 인서트 케이블(Y-cable)에도 사용됩니다.
🎯
1/4" (6.35mm) 국제 표준
프로 오디오 · 악기 · 스튜디오 장비의 표준 6.35mm 규격으로, 믹서/앰프/인터페이스의 TRS 입출력 잭에 직접 결합됩니다.
🛡️
다이캐스트 아연 합금 쉘
다이캐스트 아연 합금에 니켈 마감 처리된 금속 쉘로 내식성과 차폐 성능을 확보해 외부 간섭을 억제합니다.
✨
니켈 마감 컨택
팁·링·슬리브 컨택에 니켈 도금을 적용해 산화와 부식에 강하며, 반복 삽탈에도 접촉 안정성을 유지합니다.
🔩
솔더 타입 결선
3개 컨택 모두 솔더링 방식으로 배선되어 장기간 사용 시에도 결선부 이완이 적고, 현장에서 재작업이 용이합니다.
🔒
스트레인 릴리프
조립 후 케이블이 당겨져도 결선부에 직접 힘이 전달되지 않도록 쉘 내부의 스트레인 릴리프가 장력을 분산합니다.
