P118 SYSTEM OVERVIEW

P118은 문화·엔터테인먼트 산업의 성장과 함께 증가하는 소·중형 공연, 뮤직 페스티벌, 행사를 위해 개발된 올인원 액티브 모바일 어레이 시스템입니다. 별도의 파워앰프나 믹서 없이 단독으로 완전한 PA 시스템을 구성할 수 있습니다.

내장 믹서
BUILT-IN 4CH MIXER

별도의 믹서 없이 서브우퍼 상단 패널에서 직접 4채널 입력을 제어할 수 있습니다.

서브우퍼 상단 컨트롤 패널

1·2
GAIN A·B — Mic/Line 입력
XLR/6.35mm COMBO 소켓. LINE/MIC 선택 스위치로 라인·마이크 게인 별도 설정 가능
3
GAIN C HI-Z — 악기 입력
6.35mm JACK. 어쿠스틱 기타 등 고임피던스 악기 직접 연결
4
GAIN D — 라인 입력
RCA 스테레오 라인 입력. 음원 플레이어, 노트북 등 라인 레벨 기기 연결
DSP
DSP CONTROL
LCD 디스플레이 + 로터리 노브로 EQ·크로스오버·딜레이(0~100m, 0.25m 단위)·리미터 설정
SUB
SUB LEVEL
서브우퍼 음량을 독립적으로 조절. 저역 밸런스를 현장에 맞게 즉시 변경 가능
OUT
LINE LINK + MIX OUTPUT
LINE LINK: 입력 1·2번 신호를 개별 출력. MIX OUTPUT: 믹스 신호 출력으로 추가 스피커 연결
주요 특징
KEY FEATURES
📦
완전 독립형 PA 시스템
파워앰프, 믹서, DSP, 서브우퍼가 하나의 박스에 내장. 어레이 TOP을 꽂기만 하면 완전한 PA 시스템이 완성됩니다.
🎺
16×1" 돔 혼 HF 설계
16개의 1인치 돔 혼 HF가 100°×60° 넓은 커버리지를 제공합니다. 강한 투과력과 분석력으로 원거리까지 선명한 사운드를 전달합니다.
⚙️
24bit DSP 내장
게인, 크로스오버, EQ, 딜레이(0~100m, 0.25m 단위), 리미터, 프리셋 메모리를 LCD 패널에서 직접 설정합니다.
🔗
LINE LINK 스테레오 연결
두 대의 P118을 LINE LINK 케이블로 연결하여 스테레오 시스템을 구성할 수 있습니다. 큰 공간에서 좌우 분리된 풍부한 스테레오 음장을 제공합니다.
🔧
뉴매틱 리프팅 브라켓
공압식 리프팅 브라켓으로 어레이 TOP의 높이를 무단으로 조절합니다. TOP 내부에 이미 배선이 완료되어 있어 외부 케이블 연결이 불필요합니다.
🛞
캐스터 이동 설계
서브우퍼 하단에 캐스터(바퀴)가 장착되어 현장 이동과 포지셔닝이 매우 간편합니다. 넓은 AC 100V~240V 전원 설계로 국내외 어디서나 사용 가능합니다.
설치 방식
INSTALLATION

현장 환경에 따라 4가지 설치 방식을 선택할 수 있습니다.

🏗
기본 스탠딩
서브우퍼를 바닥에 놓고 뉴매틱 브라켓으로 어레이 TOP을 최적 높이로 세팅. 가장 빠른 기본 구성.
📐
틸팅 배치
서브우퍼를 앞쪽으로 기울여 어레이 TOP의 커버리지 방향을 조절. 계단식 객석이나 경사 무대에 적합.
🔲
사이드 필
메인 PA 시스템을 보조하는 사이드 필로 활용. MIX OUTPUT으로 메인 시스템과 신호 연동 가능.
🔗
멀티 링크
두 대의 P118을 LINE LINK 케이블로 연결하여 스테레오 구성. 큰 공간에서 좌우 분리된 음장을 제공합니다.
