내장 믹서 & DSP
BUILT-IN 4CH MIXER

서브우퍼 상단 패널에서 4채널 입력과 DSP를 직접 제어합니다. 별도 믹서 불필요.

서브우퍼 상단 컨트롤 패널 (1200A 동일)

1·2
MIC/LINE 입력 (GAIN A·B)
XLR/6.3mm COMBO 소켓. LINE/MIC 선택 스위치로 라인·마이크 게인 별도 설정. 입력 임피던스: MIC 40kΩ / LINE 10kΩ (밸런스드)
3
Hi-Z 악기 입력 (GAIN C)
6.3mm TRS 밸런스드. 어쿠스틱 기타 등 고임피던스 악기 직접 연결. 입력 임피던스: 20kΩ
4
RCA 스테레오 / 블루투스
RCA 스테레오 라인 입력. 블루투스 무선 스트리밍도 이 채널로 수신. 입력 임피던스: 5kΩ (언밸런스드)
DSP
ONE CONTROL 인코더 + 24bit DSP
프리셋 저장·불러오기, HPF, 딜레이(0~100m / 0.25m 단위), 듀얼 채널 리미터, 파라메트릭 EQ 내장
OUT
XLR 출력 + SUB LEVEL
XLR 라인 출력(1·2번) + MIX OUTPUT으로 추가 스피커 연결 가능. SUB LEVEL 노브로 저역 독립 조절
주요 특징
KEY FEATURES
⚡
600W 클래스D 바이앰핑
SUB 300W + TOP 300W 독립 CLASS-D 구성. 최대 122dB SPL로 대형 공간에서도 충분한 음량을 보장하며 발열이 낮아 장시간 운용에 최적입니다.
🔊
6×3" MF + 1" HF 컬럼
6개의 3인치 미드레인지와 1인치 컴프레션 드라이버로 악기·보컬을 선명하게 재생합니다. 수평 120°×수직 30°의 균일한 커버리지로 청중 전체에 고른 음장을 제공합니다.
🎛
4채널 내장 믹서
Mic/Line×2, RCA 스테레오, Hi-Z 악기 입력을 지원하는 4채널 아날로그 믹서 내장. 별도 믹서 없이 마이크·악기·음원 플레이어를 바로 연결할 수 있습니다.
⚙️
24bit DSP 내장
HPF, 파라메트릭 EQ, 딜레이(0~100m, 0.25m 단위), 듀얼 리미터, 프리셋 메모리. ONE CONTROL 인코더로 직관적으로 설정하거나 프리셋으로 즉시 운용합니다.
📱
블루투스 무선 스트리밍
스마트폰·태블릿에서 블루투스로 음원을 직접 전송합니다. 별도 케이블 없이 BGM을 재생하거나 음원 플레이어로 즉시 활용할 수 있습니다.
🪵
전체 우드 마감
버티컬 어레이 컬럼부터 서브우퍼 캐비닛까지 전체를 우드 마감으로 처리. 일반 플라스틱 마감과는 다른 고급스럽고 따뜻한 외관으로 인테리어 중시 공간에서도 어울립니다.
🏗
Plug & Play 자동 케이블링
컬럼을 서브우퍼 소켓에 꽂기만 하면 내부 케이블이 자동으로 연결됩니다. 공구 없이 몇 초 만에 설치 완료 — 공연 전 세팅 시간을 획기적으로 단축합니다.
설치 방식
PLUG & PLAY INSTALLATION

공구 없이 몇 초 만에 완성되는 1·2 Click 설치 방식입니다.

1
서브우퍼 배치
서브우퍼를 원하는 위치에 놓습니다.
2
컬럼 삽입
컬럼을 서브우퍼 상단 소켓에 꽂으면 내부 케이블이 자동 연결됩니다. 별도 스피커 케이블 불필요.
3
높이 조절
착탈식 컬럼 구조로 높이를 현장에 맞게 조절합니다. 최대 높이 2,118mm.
✓
전원 ON · 즉시 사용
마이크·악기·음원을 연결하거나 블루투스로 스트리밍하면 즉시 사용 가능.
