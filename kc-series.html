시리즈 소개
KC SERIES · ACTIVE COAXIAL MONITOR

동축 드라이버, Class D 앰프, 48kHz/24bit DSP를 하나의 콤팩트 캐비닛에 통합한 액티브 모니터. 스테이지 모니터링과 근접 확성 양쪽 모드를 모두 지원합니다.

KC 시리즈는 10인치(KC10D)·12인치(KC12D)·15인치(KC15D) 3개 모델로 구성되어 있습니다. 모든 모델은 LF 드라이버와 1.73" HF 컴프레션 드라이버를 동축으로 배치한 2웨이 액티브 스피커로, 동축 구조의 포인트 소스 방사를 통해 90°×90° 균일한 커버리지와 정확한 사운드 이미지를 제공합니다.

내장된 48kHz/24bit DSP는 LCD 화면을 통해 3밴드 EQ, 리미터, 딜레이, 볼륨을 직접 제어할 수 있으며, Class D 앰프(LF 300W + HF 50W)와 결합하여 고효율·경량·고출력을 동시에 실현합니다. 53° 웨지 각도로 스테이지 모니터링과 확성용 근접 사운드 시스템을 자유롭게 전환할 수 있습니다.

주요 특징
KEY FEATURES

48kHz/24bit DSP · Class D 앰프 · 동축 드라이버 · 53° 듀얼 모드까지, KC 시리즈 전 모델에 공통으로 적용되는 핵심 기술.

48kHz/24bit 내장 DSP
고해상도 48kHz/24bit DSP를 내장하여 LCD 화면을 통해 3밴드 EQ, 리미터, 딜레이, 볼륨을 직접 조절할 수 있습니다. 외부 프로세서 없이도 정밀한 사운드 튜닝이 가능합니다.
Class D 앰프
LF 300W + HF 50W의 Class D 앰프를 탑재하여 고효율과 콤팩트한 사이즈, 가벼운 무게, 안정적인 출력 성능을 동시에 실현합니다. 발열이 적고 깨끗한 사운드를 전달합니다.
동축 드라이버
10"·12"·15" LF와 1.73" HF 컴프레션 드라이버를 동축으로 배치한 포인트 소스 구조로, 90°×90°의 균일한 커버리지와 정확한 사운드 이미징을 구현합니다.
듀얼 모드 · 53° 웨지
53°의 웨지 형상 캐비닛으로 스테이지 모니터링과 근접 확성 두 가지 모드로 모두 활용할 수 있습니다. 한 대로 다양한 현장 요구를 충족합니다.
LCD 컨트롤 패널
후면 LCD 화면과 직관적인 조작계로 DSP 파라미터를 누구나 쉽게 조절할 수 있습니다. 사운드 엔지니어가 아니더라도 간편한 운용이 가능합니다.
다양한 입출력 (모델별 차등)
KC10D는 Gain A·B 두 개의 XLR-1/4" 콤보 입력, KC12D·KC15D는 Gain A RCA(스테레오)와 Gain B·C XLR-1/4" 콤보 입력 3계통을 탑재. XLR 링크 아웃을 통해 마이크·라인·스테레오 소스를 폭넓게 연결할 수 있습니다.
사용 모드
DUAL APPLICATION MODES

53° 웨지 형상 캐비닛으로 스테이지 모니터링과 근접 확성 두 가지 모드를 모두 지원합니다. 한 대의 KC 시리즈로 다양한 현장 요구에 유연하게 대응할 수 있습니다.

53°
스테이지 모니터 모드
웨지 형상으로 바닥에 눕혀 스테이지 모니터로 사용합니다. 53°의 최적화된 각도로 보컬과 연주자가 자연스럽고 정확한 모니터링 사운드를 들을 수 있습니다.
FOH
근접 확성 모드
스탠드 마운트나 직립 배치로 근접 확성용 메인 스피커로 활용합니다. 90°×90°의 넓은 커버리지로 소규모 행사장과 회의실, 강의실에 최적입니다.
