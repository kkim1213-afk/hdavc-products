CXA051

USB C-Type DAC 96kHz / 24bit · Balanced Monaural XLR · Volume / Ground Switch

CXA051은 USB C-Type 디지털 오디오 출력을 내장 DAC(96 kHz / 24bit)로 수신한 뒤, 트랜스포머를 통해 밸런스드 모노 XLR로 전달하는 패시브 신호 전송 구조의 D.I 박스입니다. 3.5mm 잭이 사라진 최신 노트북·스마트폰·태블릿을 PA·액티브 스피커·믹싱 콘솔과 직접 연결할 수 있습니다.

CXA051 · USB-C DAC PASSIVE DIRECT BOX

USB C-Type 디지털 오디오를 내장 DAC(96 kHz / 24bit)로 복조하여 밸런스드 모노 XLR로 출력하는 패시브 신호 전송 D.I 박스. 볼륨 조절 · 그라운드 스위치 · 트랜스포머 아이솔레이션.

권장 소비자가 60,000 원

Soundking CXA051은 USB-C 디지털 오디오 인터페이스 출력을 가진 스마트폰·노트북·태블릿을 전문 PA 시스템, 액티브 스피커, 믹싱 콘솔의 밸런스드 입력단과 연결하기 위해 설계된 패시브 다이렉트 박스입니다. 내장 USB-C DAC 오디오 디코딩 칩(96 kHz / 24bit)을 통해 디지털 오디오를 아날로그로 복조한 뒤, 이후 신호 전송 경로는 고품질 오디오 트랜스포머 기반의 패시브·아이솔레이티드 방식으로 처리됩니다.

전면의 볼륨 노브로 출력 레벨을 손쉽게 조정할 수 있고, 그라운딩 스위치를 통해 접지 회로에서 유발되는 노이즈를 제거할 수 있습니다. 출력 단자는 XLR 3P Male이며 밸런스드 모노 전송을 지원해 장거리 케이블링에서도 노이즈 유입이 최소화됩니다. 동일 시리즈의 CXA050(3.5mm Stereo 입력)과 컨셉을 공유하면서, 최신 USB-C 디지털 오디오 소스에 최적화된 모델입니다.

KEY FEATURES

CXA051 USB-C DAC 패시브 D.I의 핵심 포인트입니다.

🧩
USB-C DAC 디코딩 칩
USB C-Type 디지털 오디오 신호를 96 kHz / 24bit DAC 칩으로 직접 복조, 별도 오디오 인터페이스 없이 연결합니다.
🎤
Balanced Monaural XLR
XLR 3P Male 단자로 밸런스드 모노 신호를 전송해 PA 믹서·액티브 스피커와 장거리 연결 시에도 노이즈 유입이 최소화됩니다.
🎚
볼륨 조절 노브
-80 ~ -6 dBu 범위의 볼륨 감쇠를 D.I 단에서 직접 수행해 믹서 입력단 클리핑을 예방합니다.
🔇
Ground Switch
그라운딩 회로로 유입되는 노이즈를 그라운드 스위치로 차단해 현장 험 노이즈를 해결합니다.
🛡
Passive Signal Isolation
DAC 이후 신호 전송 경로는 고품질 오디오 트랜스포머 기반 패시브·아이솔레이션 방식으로 처리됩니다.
⚡
USB Bus Powered
별도 팬텀파워나 외부 어댑터 없이 USB-C 버스로부터 DAC가 작동, 현장 배선이 간단합니다.
