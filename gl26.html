GL26A-T
Active Coaxial 2-Way Full-Range Line Array Speaker

설치 방식
INSTALLATION

현장 환경에 맞게 4가지 설치 방식을 선택할 수 있습니다.

설치 모드 Overview
01
스택 (Stacking)
GL26SA-T 위에 GL26A-T를 폴로 올리는 기본 스택 구성. GL26SA 스텍키트를 사용하여 안정적으로 결합합니다.
02
서스펜션 (Hanging)
GL26SA 범퍼(플라잉 키트)를 이용해 서브우퍼와 탑을 함께 천장·트러스에 매달아 운용합니다. 3개 행잉 포인트 지원.
03
폴 마운트 (Supported)
GL26A 트레이를 35Ø 스피커 스탠드(DB075)에 장착. GL26SA-T를 바닥에 두고 탑을 폴로 올려 빠르게 셋업합니다.
04
풀 서스펜션
GL26A 범퍼와 GL26SA 범퍼를 모두 사용하여 서브우퍼와 탑 스피커를 완전히 에어리얼로 구성합니다.
주요 특징
KEY FEATURES
🎯
동축(Coaxial) 트위터 배열
8개의 트위터가 MF 유닛 위에 동축으로 배열되어 포인트 소스처럼 작동합니다. 원거리 음장 균일성과 오디오-비디오 포지셔닝 정확도가 크게 향상됩니다.
🛡
폴리우레아 캐비넷 마감
일반 파우더 코팅 대비 충격·마모·방수·UV 내구성 대폭 향상. 투어링과 야외 공연에서 캐비넷을 장기간 안전하게 보호합니다.
⚙️
내장 24비트 DSP
게인, EQ, 딜레이, 리미터, 프리셋 메모리를 패널 LCD에서 직접 조작하거나 RS-485 네트워크로 전체 시스템을 원격 통합 제어합니다.
🔌
RS-485 네트워크 통합 제어
다수의 GL26A-T / GL26SA-T를 RS-485 네트워크로 연결하여 하나의 인터페이스에서 시스템 전체를 온라인 제어합니다.
🔧
다양한 설치 유연성
스택, 서스펜션(Hanging), 폴 마운트(Supported), 풀 서스펜션 등 4가지 설치 방식. 3개 행잉 포인트로 안정적인 플라잉 운용 가능.
▶️
플러그 앤 플레이
다수의 기본 프리셋 내장으로 전문 엔지니어 없이도 즉시 운용 가능. 소규모 행사부터 중형 투어링까지 빠른 현장 셋업이 이루어집니다.
