GL26A-T

Active Coaxial 2-Way Full-Range Line Array Speaker

8개의 1인치 네오디뮴 도메 트위터와 2개의 6.5인치 네오디뮴 미드-베이스 드라이버로 구성된 풀레인지 라인어레이 스피커입니다. 트위터가 MF 스피커 위에 동축으로 배열되어 오디오·비디오 포지셔닝 정확도를 높이고 원거리 음장 균일성과 선명도를 대폭 개선합니다. 내장 고효율 Class D 앰프(400W+150W)와 24비트 DSP로 게인, 주파수 분할, EQ, 딜레이, 리미터, 프리셋 메모리를 지원합니다. RS-485 네트워크로 전체 스피커 시스템을 원격 통합 제어할 수 있습니다.

GL26SA-T

Active Single 15" Subwoofer for GL26A-T System

GL26A-T와 함께 운용하는 1,200W 고출력 15인치 액티브 서브우퍼입니다. 시스템 저역 한계를 40Hz까지 확장하여 풍성한 저역을 구현합니다. 독립형 고출력 앰프와 DSP가 탑재되어 개별 조정이 가능하며, RS-485 네트워크로 GL26A-T와 통합 제어됩니다. 정밀 서스펜션 시스템이 장착되어 다양한 설치 환경에 대응하고, 양측 매립형 손잡이로 이동·운반이 편리합니다.

왜 GL26A-T / GL26SA-T인가
KEY FEATURES

제조사 Soundking의 검증된 기술력에 현대에이브이씨의 폴리우레아 업그레이드를 더한 최적의 라인어레이 솔루션입니다.

01
🎯
동축(Coaxial) 트위터 배열
트위터가 MF 스피커 위에 동축으로 배열되어 포인트 소스처럼 작동합니다. 원거리 음장 균일성과 오디오-비디오 포지셔닝 정확도가 크게 향상됩니다.
02
🛡
폴리우레아 캐비넷 마감
일반 파우더 코팅 대비 충격·마모·방수·UV 내구성이 대폭 향상된 폴리우레아 재질로 마감합니다. 투어링 및 야외 환경에서 장비를 장기간 보호합니다.
03
⚙️
내장 24비트 DSP
게인 조정, 주파수 분할, 파라메트릭 EQ, 딜레이, 리미터, 프리셋 메모리 기능을 패널 LCD에서 직접 설정하거나 RS-485 네트워크로 원격 제어합니다.
04
🔌
RS-485 네트워크 통합 제어
다수의 GL26A-T / GL26SA-T 유닛을 RS-485 네트워크로 연결하여 하나의 인터페이스에서 전체 시스템을 온라인 제어할 수 있습니다.
05
🔧
다양한 설치 유연성
스택(stacking), 서스펜션(fly), 폴 마운트, 서포팅 등 4가지 설치 방식을 지원합니다. 3개의 행잉 포인트로 안정적인 플라잉 설치가 가능합니다.
06
▶️
플러그 앤 플레이
다수의 기본 프리셋을 내장하여 전문 엔지니어 없이도 빠른 현장 셋업이 가능합니다. 소규모 행사부터 전문 투어링까지 즉시 운용할 수 있습니다.

설치 방식
INSTALLATION
01
스택 (Stacking)
GL26SA-T 위에 GL26A-T를 폴로 올려 스택 구성하는 기본 설치 방식. GL26SA 스텍키트 사용.
02
서스펜션 (Flying)
GL26A 범퍼(플라잉 키트)와 3개의 행잉 포인트를 이용해 천장 또는 트러스에 매달아 운용.
03
폴 마운트
GL26A 트레이를 사용하여 35Ø 스피커 스탠드(DB075)에 올려 빠르고 간편하게 세팅.
04
풀 서스펜션
GL26SA-T와 GL26A-T를 모두 GL26SA 범퍼로 플라잉 구성. 완전 에어리얼 시스템 운용.
