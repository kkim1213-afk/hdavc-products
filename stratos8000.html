서브우퍼 상단 컨트롤 패널

①②
GAIN A·B — Mic/Line 입력
XLR/6.35mm COMBO 소켓. LINE/MIC 선택 스위치로 라인·마이크 게인 별도 설정
③
6.35mm JACK — 악기 입력
고임피던스 악기(어쿠스틱 기타 등) 직접 연결 가능
④
DSP CONTROL
LCD 디스플레이 + 로터리 노브로 EQ·크로스오버·딜레이(0~100m, 0.25m 단위)·리미터 설정
⑤
LINE INPUT / BLUETOOTH
RCA 스테레오 라인 입력 또는 TWS 블루투스 연결. 두 대를 TWS 모드로 페어링하여 스테레오 구성 가능
⑥⑦
MIX OUTPUT + SUB LEVEL
MIX OUTPUT: 믹스 신호 출력. SUB LEVEL: 서브우퍼 음량 독립 조절
⑧⑨
LINE LINK + INPUT SOCKET
LINE LINK: 입력 1·2번 신호 출력. INPUT SOCKET: XLR/6.35mm COMBO
주요 특징
KEY FEATURES
📦
완전 독립형 올인원 PA
파워앰프, 4채널 믹서, DSP, 15인치 서브우퍼가 하나의 서브우퍼 박스에 통합. 어레이 TOP을 스탠드에 꽂기만 하면 완전한 PA 시스템이 완성됩니다.
🔊
3×8" + 12×1" 어레이 구성
3개의 8인치 네오디뮴 풀레인지와 12개의 1인치 돔 트위터가 100°×70°의 와이드 커버리지를 제공합니다. 수직 라인 음원의 좁은 커버리지 문제를 효과적으로 해결합니다.
⚙️
24bit DSP 내장
게인, 크로스오버, EQ, 딜레이(0~100m, 0.25m 단위), 리미터, 프리셋 메모리. LCD 패널에서 직접 설정하거나 프리셋 선택으로 즉시 운용합니다.
📏
텔레스코픽 높이조절 스탠드
텔레스코픽 스탠드로 어레이 TOP의 높이를 자유롭게 조절합니다. 내부 소켓에 이미 배선이 완료되어 외부 스피커 케이블이 필요 없습니다.
🔌
플러그인 즉시 사용
어레이 TOP을 스탠드에 꽂으면 내부 소켓이 연결되어 별도 케이블 없이 바로 동작합니다. 세팅 시간을 크게 단축할 수 있습니다.
📻
TWS 블루투스
두 대의 STRATOS 8000을 TWS 블루투스 모드로 페어링하면 무선 스테레오 시스템을 구성할 수 있습니다. 케이블 없이 간편하게 넓은 공간을 커버합니다.
🔗
LINE LINK 유선 스테레오
LINE LINK 케이블로 두 대를 연결하는 유선 스테레오 구성도 가능합니다. 블루투스 또는 유선 중 현장 상황에 맞게 선택할 수 있습니다.
